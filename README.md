# TodoListAngular

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 11.2.6.

## Development server

Установка:
$ npm install
$ cd /backend
$ npm install
$ cd ..

Запуск:
$ npm start

Бэк поднимается на 1337 порте, фронт на 4200.
